代码整体结构是执行速度的决定性因素之一。性能损失与代码组织方式和具体问题解决办法直接相关。

### 循环

#### 循环性能

在 JavaScript 提供的四种循环类型(for, while, do-while, for...in)中(现在是5种了，还有一种 for...of)，只有一种循环比其他循环明显要慢：for...in 循环。除此以外，其他循环类型性能相当。
