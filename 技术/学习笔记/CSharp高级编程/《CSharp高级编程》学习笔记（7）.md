#  《C#高级编程》学习笔记（7）

## 第7章 数组

### 7.1 相同类型的多个对象

需要使用相同类型的多个对象，可以使用集合和数组。C# 用特殊的记号来声明、初始化和使用数组。

> 如果需要使用不同类型的多个对象，可以通过类、结构和元组使用它们

### 7.2 简单数组

数组是一种可以包含多个同一类型的多个元素的数据结构。

```csharp
// 声明
int[] myArray;
myArray = new int[4];
```
