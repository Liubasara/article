## 第23章 使用其他 shell

本章介绍除了 bash shell 外的其他 shell，以及介绍他们的区别。

### 23.1 dash shell

dash shell 是最简单的早期 shell —— ash 的早期优化版本，Ubuntu Linux 更是夹带私货将 sh 命令链接到了 dash 上。

与 bash shell 相比，它不具备以下性能：

- 算术运算
- test 命令
- function 命令
